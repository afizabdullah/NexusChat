# Instrucciones para Crear el Repositorio en GitHub

## Paso 1: Crear el Repositorio en GitHub

1. **Abre tu navegador** y ve a: https://github.com/new

2. **Configura el repositorio:**
   - **Repository name:** `NexusChat`
   - **Description:** (opcional) "NexusChat - Secure Messaging App with AI, WebRTC, and Tor Browser"
   - **Visibility:** Public o Private (tú decides)
   - ⚠️ **IMPORTANTE:** NO marques ninguna de estas opciones:
     - ❌ Add a README file
     - ❌ Add .gitignore
     - ❌ Choose a license

3. **Haz clic en "Create repository"**

## Paso 2: Copiar la URL del Repositorio

Después de crear el repositorio, GitHub te mostrará una página con la URL. Debería verse así:

```
https://github.com/Azelmods677/NexusChat.git
```

O si usas otro nombre de usuario:

```
https://github.com/TU_USUARIO/NexusChat.git
```

## Paso 3: Informarme

Una vez creado el repositorio, **dime la URL completa** que te mostró GitHub, y yo haré el push automáticamente.
