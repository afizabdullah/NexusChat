# 📝 Changelog

Todos los cambios notables en este proyecto serán documentados en este archivo.

El formato está basado en [Keep a Changelog](https://keepachangelog.com/es-ES/1.0.0/),
y este proyecto adhiere a [Semantic Versioning](https://semver.org/lang/es/).

## [1.0.0] - 2026-01-XX

### 🎉 Lanzamiento Inicial

#### Agregado

##### 💬 Mensajería
- Chat en tiempo real con Firebase Realtime Database
- Mensajes de voz con visualización de forma de onda animada
- Compartir multimedia: imágenes, videos y archivos
- Indicadores de estado: enviado / entregado / leído
- Indicador de escritura en tiempo real
- Responder mensajes deslizando a la derecha
- Reacciones con emojis manteniendo pulsado
- Chats grupales con gestión de miembros
- Fijar, silenciar y archivar conversaciones
- Búsqueda de mensajes y conversaciones

##### 📖 Historias
- Caducidad automática a las 24 horas
- Barra de progreso con animación suave
- Reacciones y respuestas rápidas
- Lista de visualizaciones con marcas de tiempo
- Anillos animados para estados
- Soporte multimedia: imágenes, videos y texto
- Selector completo de emojis con 4 categorías
- Emojis draggables con posicionamiento libre
- Ajuste bidimensional de fotos (X+Y)
- Texto draggable sobre fondo negro

##### 📞 Llamadas
- Llamadas de voz con WebRTC
- Videollamadas con calidad HD
- Comunicación P2P nativa
- Calidad adaptativa según condiciones de red
- Controles: silenciar, altavoz, cambiar cámara
- Permisos actualizados (FOREGROUND_SERVICE_PHONE_CALL, MANAGE_OWN_CALLS)

##### 🤖 Azel IA
- Asistente de IA avanzado sin censura
- 10 modelos disponibles (DeepSeek R1, Llama 3.3, Qwen, Mistral, etc.)
- Streaming en tiempo real
- Contexto extendido (40 mensajes)
- Hasta 16,384 tokens por respuesta
- Especializado en hacking, programación y ciberseguridad
- Historial de conversaciones persistente
- Análisis de seguridad de código
- Generación de payloads
- Consultas de técnicas de hacking

##### 🎨 Personalización
- 15 temas de color predefinidos
- Fondos personalizados: imagen, video, color o degradado
- Fondos independientes por conversación
- Burbujas de mensaje con efectos visuales
- Tamaños de fuente ajustables
- Modo oscuro optimizado
- Navegación por gestos entre secciones

##### 🔐 Seguridad
- Cifrado E2EE con Signal Protocol
- Navegador Tor integrado
- Gestión de claves criptográficas
- Autenticación Firebase (Email + Google)
- Almacenamiento seguro de archivos
- Sistema de respaldo cifrado

##### 🔧 Herramientas
- Terminal emulador completo
- Editor de código Sora Editor
- Generador de payloads
- Análisis de vulnerabilidades
- Automatización de tareas

##### 🔔 Notificaciones
- Agrupadas por conversación
- Respuesta rápida desde el panel
- Marcar como leído sin abrir la app
- Avatar del remitente
- Firebase Cloud Messaging

#### Tecnologías

- Jetpack Compose BOM 2025.04.01
- Kotlin 100%
- Firebase BOM 33.7.0
- Hilt 2.52
- WebRTC 1.1.3
- Signal Protocol 0.40.1
- Coil 3.1.0
- ExoPlayer Media3 1.5.1
- Room Database
- DataStore 1.1.1
- Coroutines 1.9.0
- OkHttp 4.12.0
- BouncyCastle 1.78.1
- Sora Editor 0.23.5
